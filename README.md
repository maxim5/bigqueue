# BigQueue

A fork of [original BigQueue](https://github.com/bulldog2011/bigqueue/), extended and productionized.

Important: this library is *not compatible* with original, but the migration is straightforward.

## Gradle Setup

```kotlin
repositories {
    maven("https://jitpack.io")
}

dependencies {
    implementation("com.github.maxim5:bigqueue:0.1.0")
}
```
